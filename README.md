# Password Strength Checker

## Overview

This is a Python-based Password Strength Checker that analyzes a password and classifies it as Weak, Medium, or Strong.

The program evaluates passwords using common cybersecurity best practices such as:

* Minimum length requirement
* Uppercase letters
* Lowercase letters
* Numbers
* Special characters

It also provides suggestions to improve password security.

## Features

* Password strength analysis
* Strength scoring system
* Security recommendations
* User-friendly terminal interface
* Built using Python
